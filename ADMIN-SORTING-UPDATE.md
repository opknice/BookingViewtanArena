## 📝 Logic การเรียงลำดับใหม่:

### 1. **เรียงตามสถานะ** (Priority สูงสุด)
```
🟡 Pending    (Priority: 0) → แสดงก่อนสุด
🟢 Confirmed  (Priority: 1) → แสดงตรงกลาง  
🔴 Cancelled  (Priority: 2) → แสดงท้ายสุด
```

### 2. **เรียงตามวันที่** (ถ้าสถานะเดียวกัน)
```
วันที่ใหม่ → เก่า
(2024-12-25 → 2024-12-24 → 2024-12-23)
```

### 3. **เรียงตามเวลา** (ถ้าวันเดียวกัน)
```
เวลาเร็ว → สาย
(16:00 → 17:00 → 18:00)
```

## 🔄 เปรียบเทียบ:

### ❌ เดิม (ก่อนแก้ไข):
```javascript
.sort((a, b) => {
  const aFirst = a[0]
  const bFirst = b[0]
  // เรียงตามวันที่อย่างเดียว
  if (aFirst.date !== bFirst.date) return bFirst.date.localeCompare(aFirst.date)
  return aFirst.startTime.localeCompare(bFirst.startTime)
})
```

**ผลลัพธ์:** แสดงตามวันที่ ไม่สนใจสถานะ
```
1. Confirmed - 2024-12-25
2. Pending   - 2024-12-24   ❌ Pending ไม่ได้อยู่บนสุด
3. Cancelled - 2024-12-23
```

---

### ✅ ใหม่ (หลังแก้ไข):
```javascript
.sort((a, b) => {
  const aFirst = a[0]
  const bFirst = b[0]
  
  // 1. เรียงตามสถานะก่อน
  const aStatus = groupStatus(a)
  const bStatus = groupStatus(b)
  const statusPriority = { pending: 0, confirmed: 1, cancelled: 2 }
  
  if (aStatus !== bStatus) {
    return statusPriority[aStatus] - statusPriority[bStatus]
  }
  
  // 2. ถ้าสถานะเดียวกัน เรียงตามวันที่
  if (aFirst.date !== bFirst.date) return bFirst.date.localeCompare(aFirst.date)
  
  // 3. ถ้าวันเดียวกัน เรียงตามเวลา
  return aFirst.startTime.localeCompare(bFirst.startTime)
})
```

**ผลลัพธ์:** แสดง Pending ก่อนเสมอ
```
1. 🟡 Pending   - 2024-12-24  ✅ Pending อยู่บนสุดเสมอ!
2. 🟢 Confirmed - 2024-12-25
3. 🔴 Cancelled - 2024-12-23
```

## 🔧 Technical Details:

### ฟังก์ชันที่เกี่ยวข้อง:

```javascript
// 1. Group orders by groupId
const groupOrders = (ordersList) => {...}

// 2. Determine group status
const groupStatus = (group) => {
  if (group.some((order) => order.status === 'pending')) return 'pending'
  if (group.some((order) => order.status === 'confirmed')) return 'confirmed'
  return 'cancelled'
}

// 3. Sort with priority
statusPriority = { pending: 0, confirmed: 1, cancelled: 2 }
```
